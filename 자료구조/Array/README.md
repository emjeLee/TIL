# 배열  
-   여러 개체 값을 순차적으로 나열한 선형 자료 구조  

# 속성 및 메서드  

- 배열 크기 및 배열 여부 확인: length(), isArray()  
- 배열 추가/삭제 : push(), pop(), shift(), unshift(), splice(), slice()  
- 배열 탐색: indexOf(), lastIndexOf(), includes()  
- 배열 변형: sort(), resverse(), join()  
- 배열 반복: forEach(), map(), find(), filter(), reduce()  
- 논리 연산: some(), every()
